# Inventory Management System

Excel-based inventory tracking system with automated stock calculations, 
conditional formatting, data validation, and interactive dashboard.

## Features
•⁠  ⁠Automated stock level tracking using SUMIF formulas
•⁠  ⁠Real-time reorder alerts via conditional formatting (red/green)
•⁠  ⁠Dropdown data validation for Category and Supplier fields
•⁠  ⁠Pivot Table & Pivot Chart for category-wise stock value analysis
•⁠  ⁠Interactive KPI dashboard (Total Items, Low Stock Items, Total Value)

## Tools Used
Excel, SUMIF, INDEX-MATCH, Conditional Formatting, Data Validation, Pivot Tables
